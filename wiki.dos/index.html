Welcome to dOS0

dOS0 is a web-based desktop operating system simulator featuring a complete windowing environment, file system with cloud synchronization, and its own isolated internet ecosystem called dOS0 Web.

Note: You are currently browsing wiki.dos, a .dos website within the dOS0 Web ecosystem. This is a simulated internet environment separate from the external web.

Core Features

Desktop Environment

A fully functional desktop interface with draggable, resizable windows, a taskbar, and system tray. The environment supports multiple concurrent applications and window management operations including minimize, maximize, and close.

File System

Hierarchical file system with support for files and folders. All data is persisted using GitHub as a cloud storage backend, enabling cross-device synchronization and version control capabilities.

Application Suite

Built-in applications include:

  • File Explorer - Navigate and manage your file system
  • Notepad - Text editor with syntax highlighting support
  • Calculator - Scientific calculator with unit conversion
  • Browser - Access .dos websites within the dOS0 Web
  • Settings - Customize appearance and system configuration

dOS0 Web

An isolated internet environment where users can browse .dos websites. Unlike traditional web browsers, the dOS0 Browser cannot access external internet resources, creating a self-contained web ecosystem.

Quick Start

To begin using dOS0:

  1. Configure GitHub authentication in Settings to enable cloud sync
  2. Create folders and files using File Explorer
  3. Open the Browser to explore available .dos websites
  4. Customize your desktop theme and background in Settings

Documentation

Use the sidebar navigation to explore detailed documentation for each component of the system. Topics include application guides, file system operations, GitHub synchronization setup, and .dos website development.
